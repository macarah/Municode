CODE COMPARATIVE TABLE - ORDINANCES AND RESOLUTIONS
modified
This table gives the location within this Code of those ordinances and resolutions adopted since the 1989 Code, as updated through April 8, 1996, which are included herein. Ordinances and resolutions adopted prior to such date were incorporated into the 1989 Code, as supplemented. This table contains some ordinances and resolutions which precede April 8, 1996, but which were never included in the 1989 Code, as supplemented, for various reasons. Ordinances and resolutions not listed herein have been omitted as repealed, superseded or not of a general and permanent nature.
