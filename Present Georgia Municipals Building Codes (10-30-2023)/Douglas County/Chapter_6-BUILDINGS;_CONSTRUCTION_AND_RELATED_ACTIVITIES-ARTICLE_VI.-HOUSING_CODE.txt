ARTICLE VI. - HOUSING CODE

Sec. 6-60. - Code adopted.
There is hereby adopted for the purpose of establishing standards and procedures affecting the habitability and sanitary conditions of private premises and structures, the Standard Housing Code, latest edition, being particularly that code published by the Southern Building Code Congress International, Inc., 900 Montclair Road, Birmingham, Alabama 35213-1206, and said code is adopted and incorporated as fully as if set out at length herein. At least one (1) copy of the housing code is on file in the office of the building official.
(Ord. of 9-17-91; Ord. of 8-19-97)
