ARTICLE VIII. - SWIMMING POOL CODE

Sec. 6-90. - Code adopted.
There is hereby adopted for the purpose of establishing standards and procedures affecting the design, erection and construction of building and structures, the Standard Swimming Pool Code, latest edition, being particular that code published by the Southern Building Code Congress International, Inc., 900 Montclair Road, Birmingham, Alabama 35213-1206, and said code is hereby adopted and incorporated as fully as if set out at length herein. At least one (1) copy of the code is on file in the office of the building official.
(Ord. of 9-17-91; Ord. of 8-19-97)
Sec. 6-91. - Permit required.
It shall be unlawful for any person to construct a swimming pool until a building permit has been obtained from and the required fee paid to the permit office.
