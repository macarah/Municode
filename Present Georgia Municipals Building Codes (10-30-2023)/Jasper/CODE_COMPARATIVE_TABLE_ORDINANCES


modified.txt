This table gives the location within this Code of those ordinances adopted since the 1979 Code, as updated through November 20, 1994, that are included herein. Ordinances adopted prior to such date were incorporated into the 1979 Code, as supplemented. This tables contains some ordinances which precede November 20, 1994, but that were never included in the 1979 Code, as supplemented, for various reasons. Ordinances adopted since November 20, 1994, and not listed herein, have been omitted as repealed, superseded or not a general and permanent nature.
