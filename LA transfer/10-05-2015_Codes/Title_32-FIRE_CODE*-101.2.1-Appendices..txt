101.2.1 - Appendices.
Provisions in the appendices shall not apply unless specifically adopted. Only Appendix B, Appendix BB, Appendix C, Appendix CC, and Appendix K from the 2013 California State Fire Code have been incorporated into this code. This code also adopts Appendix L and Appendix M.
