111.1 - Order.
Whenever the fire code official finds any work regulated by this code being performed in a manner contrary to the provisions of this code or in a dangerous or unsafe manner, the fire code official is authorized to issue a stop work order. The fire code official is authorized to order the work, the operation, or the use stopped or the evacuation of any premises, building, or vehicle or portion thereof which has or is a condition hazardous to life or property.
