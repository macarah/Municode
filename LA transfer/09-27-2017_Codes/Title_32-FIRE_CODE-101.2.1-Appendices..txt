101.2.1 - Appendices.
Provisions in the appendices shall not apply unless specifically adopted. Only Appendix B, Appendix BB, Appendix C, and Appendix CC, from the 2016 California State Fire Code have been incorporated into this code. This code also adopts Appendix D, Appendix N, Appendix O, Appendix P, and Appendix Q.
