105.4 - Code Enforcement Appeals Board.
In order to conduct the hearings provided for in Section 103.4.5 and Government Code Section 54988, there shall be a Code Enforcement Appeals Board. The Code Enforcement Appeals Board shall be comprised of the same five members and alternates as the Building Rehabilitation Appeals Board. The Building Official shall be an ex officio nonvoting member and act as secretary.
